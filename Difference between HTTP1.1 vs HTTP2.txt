Difference between HTTP1.1 vs HTTP2


HTTP1.1	

1.The first version of HTTP was called HTTP/1.1.	
2.It works on the textual format.	
3.It can load one requests at a time. Hence one request per one TCP connection is possible.	
4.It compresses data by itself.

HTTP2
1.A new version of HTTP was called HTTP/2.
2.It works on the binary protocal.
3.It uses single TCP connection to deliver multiple requests/responses.
4.It uses HPACK for data compression.
